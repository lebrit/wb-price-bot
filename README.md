# WB Price Bot

Self-hosted Telegram-бот для отслеживания цен Wildberries. Поддерживает много товаров и
несколько разрешённых пользователей, уведомления по проценту, сумме или целевой цене,
историю изменений, наличие и экспериментальную персональную цену своего WB-аккаунта.

Текущая версия: `0.1.0`.

## Что уже работает

- добавление товара по полной ссылке, короткой ссылке WB или артикулу;
- отдельный список товаров каждого Telegram-пользователя;
- условия «подешевеет на X%», «подешевеет на X ₽» и «станет не дороже X ₽»;
- уведомление при возвращении товара в наличие;
- история только реальных изменений цены/наличия;
- надёжная очередь уведомлений в SQLite с повторной отправкой после ошибок Telegram;
- пауза, удаление и ручная проверка отдельного товара;
- пакетная проверка публичных цен, ограничение частоты и случайный разброс времени;
- явная подпись источника: публичная цена, цена аккаунта или цена с WB Кошельком;
- зашифрованное хранение браузерной сессии аккаунта (Fernet);
- интерактивное русское меню установки, обновления, управления, диагностики,
  резервного копирования и удаления;
- один непривилегированный Docker-контейнер без открытых входящих портов;
- зафиксированные версии runtime-зависимостей в `requirements.lock`;
- SQLite в WAL-режиме и согласованные резервные копии через SQLite Backup API.

## Важное ограничение Wildberries

У Wildberries нет документированного API покупательских и персональных цен: официальный
[WB API](https://dev.wildberries.ru/en/openapi/api-information?locale=ru) предназначен
продавцам. Публичный адаптер использует недокументированный endpoint витрины, а персональный
режим — обычную карточку товара в авторизованном Chromium.

Действующая [оферта Wildberries для покупателей](https://legal.wildberries.ru/consumers-offer/country/ru/lang/ru/)
прямо запрещает использовать программы, алгоритмы, скрипты, краулеры и ботов для сбора
информации, включая цены и наличие, и допускает блокировку кабинета. Консервативный интервал
не устраняет это нарушение. Для эксплуатации нужен письменный допуск Wildberries или
лицензированный источник данных; без него проект следует считать исследовательским
прототипом с юридическим риском. Endpoint может измениться, начать отвечать `403/429/498`, а
сессия — истечь. Проект не обходит CAPTCHA и другие проверки и не выполняет покупки,
операции с корзиной или платежами.

Цена в уведомлении не гарантирует итоговую сумму заказа: на неё влияют регион, размер,
продавец, логистика, способ оплаты, WB Кошелёк и активные акции.

## Быстрая установка на Ubuntu/Debian

Нужно заранее:

1. Создать бота у [@BotFather](https://t.me/BotFather) и получить токен.
2. Узнать числовой Telegram ID владельца.

Одна команда открывает интерактивное меню с пунктом «Установить»:

```bash
(umask 077; S=$(mktemp); trap 'rm -f "$S"' EXIT; curl -fsSL https://raw.githubusercontent.com/lebrit/wb-price-bot/v0.1.0/scripts/install.sh -o "$S" && bash -n "$S" && sudo bash "$S")
```

Установщик сам:

- поставит Docker/Compose из системного репозитория, если их нет;
- спросит Telegram token, разрешённые ID, интервал и региональный `dest`;
- создаст отдельные каталоги данных и секретов;
- соберёт контейнер с Chromium;
- проверит Telegram и дождётся здорового состояния сервиса;
- установит команду `wb-price-bot`.

Полный upgrade ОС, перезагрузка, открытие портов и удаление чужих Docker-ресурсов не
выполняются.

После установки:

```bash
sudo wb-price-bot
```

## Интерактивное меню

Главное меню содержит:

1. обновление до последнего GitHub Release с backup и rollback;
2. запуск, остановку, перезапуск и состояние;
3. просмотр журнала;
4. замену/проверку Telegram token, allowlist ID и тестовое сообщение;
5. инструкцию, импорт и удаление WB-сессии;
6. интервал мониторинга, региональный `dest` и немедленную проверку;
7. создание, просмотр и восстановление резервных копий;
8. диагностику Docker, SQLite, прав секретов, диска и Telegram;
9. удаление только runtime либо полный purge с разными контрольными фразами.

Явные команды также доступны:

```bash
sudo wb-price-bot status
sudo wb-price-bot logs
sudo wb-price-bot backup
sudo wb-price-bot doctor
sudo wb-price-bot update
```

## Использование бота

Основные команды:

- `/add` — добавить товар;
- `/list` — товары, история, пауза, удаление и ручная проверка;
- `/account` — персональная цена своего аккаунта (beta);
- `/status` — состояние мониторинга;
- `/cancel` — отменить текущий мастер.

Для порога по проценту или сумме цена при добавлении становится контрольной. Если товар
дорожает, контрольная цена повышается. Когда накопленное падение достигает порога, бот
уведомляет и начинает считать следующее падение от новой цены. Так постепенное снижение не
теряется и одинаковая цена не вызывает повторные сообщения.

## Подключение своего аккаунта Wildberries

SMS-код и CAPTCHA нельзя безопасно передавать Telegram-боту. Вход выполняется вручную в
отдельном обычном браузере на вашем компьютере:

```bash
git clone https://github.com/lebrit/wb-price-bot.git
cd wb-price-bot
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/macOS: source .venv/bin/activate
python -m pip install -e ".[browser]"
python -m playwright install chromium
python scripts/capture_wb_session.py
```

В открывшемся окне войдите в Wildberries, выберите адрес доставки и откройте любую карточку.
Помощник сохранит `wb-session.json`. Не отправляйте его в Telegram: обычные чаты ботов не
являются Secret Chats. Передайте файл прямо своему серверу по SSH, без промежуточной записи:

```bash
ssh USER@SERVER "sudo wb-price-bot wb-session-import ВАШ_TELEGRAM_ID" < wb-session.json
```

Альтернатива — скопировать файл по SCP, открыть `sudo wb-price-bot` → «Аккаунт
Wildberries» → «Импортировать» и сразу удалить исходный файл с сервера.

Не отправляйте сессию другим людям, в мессенджеры и не добавляйте её в Git. Если проверка авторизации
сломалась, бот приостанавливает персональный режим и сообщает об этом; он не создаёт ложное
изменение цены скрытым переходом на гостевой режим.

Подробнее: [docs/WB_PROVIDER.md](docs/WB_PROVIDER.md) и [docs/SECURITY.md](docs/SECURITY.md).

## Данные и каталоги сервера

```text
/opt/wb-price-bot/releases/       неизменяемые релизы
/opt/wb-price-bot/current         активный релиз
/etc/wb-price-bot/config.env      несекретные настройки
/etc/wb-price-bot/secrets/        Telegram token и ключ шифрования
/var/lib/wb-price-bot/            SQLite и внутренние backup
/var/backups/wb-price-bot/        операторские backup
/usr/local/bin/wb-price-bot       команда управления
```

Telegram token и ключ сессии не включаются в переносимую копию базы. Для восстановления на
другом сервере ключ шифрования нужно переносить отдельно по защищённому каналу либо заново
подключать WB-аккаунт.

## Локальная разработка

```bash
python -m venv .venv
.venv\Scripts\activate
python -m pip install -e ".[dev]"
ruff check .
ruff format --check .
mypy
pytest
```

Локальный запуск:

```bash
set TELEGRAM_BOT_TOKEN=...
set SESSION_ENCRYPTION_KEY=...
set TELEGRAM_ALLOWED_USERS=123456789
python -m wb_price_bot run
```

Ключ Fernet можно создать командой `python -m wb_price_bot generate-key`.
