# Провайдер Wildberries

Состояние проверено 10 июля 2026 года. Официального buyer API нет; схема может измениться.

## Публичный режим

Используется `https://card.wb.ru/cards/v4/detail` с `nm` через `;`, валютой, языком и
региональным `dest`. Цена хранится в копейках и считается как `product + logistics`.
Выбирается минимальная цена среди доступных размеров; если доступных нет — известная цена
сохраняется, но товар отмечается отсутствующим. Ноль, `null`, пропавший товар и ошибочный
JSON никогда не считаются снижением.

По умолчанию одна пачка содержит не более 25 артикулов, интервал — 30 минут, jitter — две
минуты. Если встроенный HTTP-клиент получает `403/498` из-за сетевого профиля, тот же
публичный endpoint один раз запрашивается системным `curl`. Если и он получает защитный
ответ, проверка прекращается: challenge не решается и цена не подменяется.

## Аккаунтный режим (beta)

`scripts/capture_wb_session.py` открывает видимый Chrome/Chromium. Пользователь выполняет
штатный вход WB ID и любые защитные проверки сам. Playwright экспортирует cookies,
localStorage и IndexedDB в storage state; после импорта он шифруется.

На сервере Chromium открывает карточку обычным способом. Адаптер использует ответ страницы
для метаданных и видимую цену DOM, включая WB Кошелёк, если она действительно показана.
Наличие Bearer-запроса страницы служит признаком активной авторизации. Без него проверка
останавливается с ошибкой; гостевая цена не подставляется.

Цена зависит от браузерного региона/ПВЗ. При входе нужно выбрать тот же адрес, который
используется для покупок.

## Известные ограничения

- пока отслеживается минимальная цена доступного размера, а не конкретный `optionId`;
- DOM-селекторы и endpoint внутренние;
- браузерный режим заметно тяжелее пакетного публичного запроса;
- финальная checkout-цена может отличаться;
- действующая оферта Wildberries прямо запрещает автоматический сбор цен и наличия; для
  эксплуатации нужен письменный допуск или лицензированный источник.
